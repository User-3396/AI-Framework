California Housing Predictor

Estime o valor mediano das casas na Califórnia a partir de atributos geográficos, populacionais e socioeconômicos.

Resumo

Este sistema utiliza um modelo de regressão para estimar o valor mediano das casas em uma região da Califórnia.

  • Dados geográficos
  • Dados populacionais
  • Dados econômicos
  • Categoria de proximidade com o oceano
O resultado representa o valor mediano previsto para o distrito, não o preço exato de uma casa individual.
